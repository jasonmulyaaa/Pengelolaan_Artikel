#################################################
Pengelolaan Artikel dengan Codeigniter and Vue JS
#################################################

1. PHP version : 8.0.12
2. Codeigniter version: 4
3. Vue JS version: v3.3.4

********
Feature:
********
1. Create
2. Read
3. Edit
4. Delete
5. Search
6. Views Count

********
Include:
********
1. Bootstrap 5 - CSS
2. Axios - AJAX/HTTP library

********
Running:
********
1. Codeigniter(backend) -> php spark serve
2. Vue Js(frontend) -> npm run dev

********
Database:
********
1. Create db "pengelolaan_artikel" in mysql
2. Import pengelolaan_artikel.sql
